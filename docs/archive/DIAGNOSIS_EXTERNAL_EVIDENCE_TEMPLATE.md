# 数据库诊断外部证据模板

## 1. 使用说明

本模板用于补齐仓库外部的生产证据, 包括日志上下文, Grafana 截图, 抓包, strace, EXPLAIN 输出, 以及四方签字记录。

说明：

- 本文件为模板，默认包含示例值与待填字段；请复制并另存为实际证据文件后再填写。
- 如证据不落库，请在“截图链接/输出文件”处填写内部系统链接，并确保保留期 ≥90 天。
- 门禁要求：必须提交 `docs/db/DIAGNOSIS_EXTERNAL_EVIDENCE_*.md`（不含 TEMPLATE），且不得包含“待补充/待签字/待审批/待评审/占位/示例”等占位词，并至少包含一个 http(s) 证据链接。
