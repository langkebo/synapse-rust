## 使用指南

### 1. 开始优化工作前

1. 阅读 `compliance-report.md` 了解当前问题
2. 阅读 `optimization-plan.md` 了解修复方案
3. 使用 `checklist.md` 跟踪进度

### 2. 执行优化工作时

1. 按优先级顺序处理问题
2. 完成一项后在 `checklist.md` 中更新状态
3. 遇到问题及时更新风险评估

### 3. 完成优化工作后

1. 执行测试验证
2. 更新相关文档
3. 进行验收确认
