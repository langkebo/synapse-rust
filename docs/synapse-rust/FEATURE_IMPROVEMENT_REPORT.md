### 4.2 联邦功能配置

**需要配置的环境变量**:

```bash
# 联邦签名密钥
export FEDERATION_SIGNING_KEY="your-signing-key"

# 联邦服务器名称
export SERVER_NAME="your-server.com"

# 联邦目标服务器
export FEDERATION_TARGET="target-server.com"
```

**配置步骤**:

1. 生成联邦签名密钥
2. 配置服务器名称
3. 设置联邦目标服务器
4. 重新运行联邦测试

2. 对于向后兼容的API，支持多版本
